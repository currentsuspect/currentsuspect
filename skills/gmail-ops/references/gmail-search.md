# Gmail Search Operators (Quick Reference)

## Categories
- category:promotions
- category:social
- category:updates
- category:forums

## Date / age
- older_than:2y
- older_than:1y
- older_than:6m
- before:2024/02/14
- after:2024/02/14

## Size
- larger:10m
- larger:20m
- larger:50m

## Attachments
- has:attachment
- filename:pdf
- filename:zip

## Status
- is:unread
- is:read
- is:starred
- is:important
- is:spam

## Examples
- Promotions older than 2 years: `category:promotions older_than:2y`
- Any mail older than 2 years: `older_than:2y`
- Large attachments (report only): `has:attachment larger:10m`
- Spam cleanup: `is:spam`
