## Azure cost investigation (Feb 15 morning)
- Dylan noticed ~$2/day Azure cost regardless of VM on/off.
- OS disk is `Premium_LRS` (Premium SSD) P6 64GB — costs ~$9.60/mo vs Standard SSD ~$4.80/mo.
- Premium SSD provides no real benefit at 64GB size (same IOPS/throughput as Standard SSD at that tier).
- Recommended switching to `StandardSSD_LRS` to save ~$5/month. Requires VM deallocation → disk SKU change → restart.
- Full Azure cost breakdown from Feb 13: VM ~$32.5/mo, static IP ~$3.6/mo, P6 disk ~$11.1/mo; total ~$47-48/mo.
