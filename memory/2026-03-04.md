
## Curated Consciousness Forum — Literary Project

**Project:** Built "Curated Consciousness" — a fictional forum website for a literary novel about radicalization. A men's community that appears to be about self-improvement but gradually reveals itself as a manipulation-doctrine community.

**Live Site:** https://curated-consciousness.vercel.app

**Tech Stack:**
- React 18 + TypeScript
- Tailwind CSS with dark mode
- Framer Motion for animations
- Vite for build
- Vercel for deployment

**Key Features Built:**
- Homepage with community description (subtle red flags)
- 5 forum categories with varying restriction levels
- 3 full thread pages with complete post content
- Sidebar with community rules and wiki links
- Dark/light mode toggle
- Mobile-responsive design
- Wiki system with expandable sections
