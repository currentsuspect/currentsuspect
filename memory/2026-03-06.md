### 2. OfflineRenderRegressionTest (Tests/Headless/OfflineRenderRegressionTest.cpp)
Regression test harness:
```bash
OfflineRenderRegressionTest project.aes reference.wav [options]
  --duration-seconds N    (default: 5)
```

**Metrics:**
- RMS difference (dB)
